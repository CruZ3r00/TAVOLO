Configurazione iniziale

pos-rt-service

Collega questo dispositivo al tuo account Strapi per abilitare stampa e pagamenti.

Recupera il codice dal file .claim-code nella cartella dati del servizio (Windows: %ProgramData%\PosRtService; macOS: ~/Library/Application Support/PosRtService; Linux: ~/.local/share/pos-rt-service).

Genera il token dalla pagina profilo del tuo account Strapi → "POS / Cassa fiscale". È usa-e-getta e scade in 30 minuti.
